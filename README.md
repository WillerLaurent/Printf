# Projet ft_printf

## Sujet
Recoder la fonction printf de la libc.

### Partie obligatoire
#### Types : c, s, p, d, i, u, x, X, %

#### Indicator : 0, -
#### Width
#### Precision

### Bonus
#### Types : n, f, g, e

#### Indicator : # ' ' +
#### Taille : l, ll, h, hh
